SSD-TEMP-WARN

SSD operating temperature warning

Medium Severity Category: Storage Errors (SSD/HDD)

Description

The NVMe or SATA SSD is operating above optimal temperatures.

Common Causes

  • Poor case airflow
  • Missing M.2 heatsink

Recommended Solutions

  • Solution: Install an M.2 heatsink
  • Solution: Improve chassis ventilation

Diagnostic Commands

  • wmic diskdrive get status
